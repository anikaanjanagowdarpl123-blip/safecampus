# 🛡️ SafeCampus – Disaster Preparedness System

A web application that helps students and staff quickly learn emergency response procedures through scenario-based guidance, safety checklists, and quick-access emergency resources.

## 🔗 Live Demo
[Click here to view the live app](https://YOURUSERNAME.github.io/safecampus)

## 🚀 Features
- 🌍 Disaster Learning Modules (Earthquake, Fire, Flood, Cyclone)
- ⚡ Scenario-Based Step-by-Step Guidance
- ✅ Interactive Preparedness Checklist with Score
- 📞 Emergency Contacts with Direct Call & SMS
- 🗂️ JSON Schema Output
- 📱 Mobile-Friendly Responsive Design

## 🗂️ Project Structure
```
safecampus/
├── index.html       # Main UI and layout
├── style.css        # All styling and responsive design
├── app.js           # Core logic, disaster data, JSON schema
├── requirements.txt # Project dependencies
└── README.md        # Project documentation
```

## 🛠️ Tech Stack

- CSS3
- Vanilla JavaScript
